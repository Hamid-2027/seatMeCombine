Bus Companies
* POST /api/bus-companies - Create a new bus company.
* GET /api/bus-companies - Get all bus companies.
* GET /api/bus-companies/:id - Get a specific bus company.
* PUT /api/bus-companies/:id - Update a bus company.
* DELETE /api/bus-companies/:id - Delete a bus company.

User Profiles
* POST /api/user-profiles - Create a new user profile.
* GET /api/user-profiles - Get all user profiles.
* GET /api/user-profiles/:id - Get a specific user profile.
* PUT /api/user-profiles/:id - Update a user profile.
* DELETE /api/user-profiles/:id - Delete a user profile.
* POST /api/user-profiles/:userId/passengers - Add a saved passenger to a user profile.
* PUT /api/user-profiles/:userId/passengers/:passengerId - Update a saved passenger.
* DELETE /api/user-profiles/:userId/passengers/:passengerId - Delete a saved passenger.
* POST /api/user-profiles/:userId/payment-methods - Add a saved payment method to a user profile.
* PUT /api/user-profiles/:userId/payment-methods/:pmId - Update a saved payment method.
* DELETE /api/user-profiles/:userId/payment-methods/:pmId - Delete a saved payment method.
